# Дизайн-система Personal Cosmic Book: четыре стихии

## Решение

Продукт использует **одну дизайн-систему и четыре фиксированные темы**, связанные с классическими стихиями. Это не четыре разных продукта и не уникальная генерация дизайна для каждой книги.

Темы:

1. `earth` — **Земля · Корни**;
2. `water` — **Вода · Глубина**;
3. `air` — **Воздух · Дыхание**;
4. `fire` — **Огонь · Искра**.

## Продуктовый смысл

Связь со стихиями делает выбор понятным пользователю и даёт четыре выразительных визуальных настроения. При этом тема является оформлением, а не психологическим диагнозом и не отдельным эзотерическим выводом.

Пользователь может выбрать тему сам. Автоматический режим назначает тему детерминированно по рассчитанному element profile.

## Формула вариативности

```text
1 общий layout
× 4 темы
× 4 фиксированных символа стихий
× персональное имя и название
× детерминированная cosmic signature из book_id
```

Убирается библиотека из 12 архетипических обложек: основной символ совпадает с выбранной стихией. Это сокращает число комбинаций, визуальную QA и стоимость поддержки.

## Что всегда одинаково

- сетка страниц;
- типографическая шкала;
- набор шрифтов;
- структура обложки;
- структура десяти разделов;
- компоненты цитат, выводов, практик и списков;
- оглавление и навигация;
- адаптивность;
- PDF/print styles;
- правила page break;
- доступность и минимальный контраст.

## Что меняется между темами

- цветовые токены;
- символ стихии;
- фоновой SVG-паттерн;
- форма разделителя;
- один набор декоративных линий;
- интенсивность декоративных элементов в пределах общего layout.

## Что персонализируется в каждой книге

- имя;
- название и подзаголовок;
- выбранная или назначенная тема;
- короткая подарочная надпись;
- акцентные цитаты из текста;
- детерминированная cosmic signature из `book_id`.

## Тема 1: Земля · Корни — `earth`

**Ощущение:** опора, зрелость, телесность, спокойная сила.

**Мотивы:** слои породы, контурные линии, круги роста, мягкая фактура бумаги. Не использовать буквальные клипарты листьев, гор и кристаллов.

```json
{
  "cover_background": "#293029",
  "cover_text": "#F7F0E3",
  "content_background": "#F7F1E7",
  "primary": "#66704F",
  "secondary": "#A66A4A",
  "text": "#2F2B26",
  "metal": "#B9975B",
  "pattern": "strata",
  "symbol": "earth"
}
```

## Тема 2: Вода · Глубина — `water`

**Ощущение:** глубина, чувствительность, текучесть, внутренняя жизнь.

**Мотивы:** волновые линии, круги на воде, мягкий туман, отражения. Не использовать буквальные изображения моря, русалок и капель как главный декор.

```json
{
  "cover_background": "#102A3A",
  "cover_text": "#F2F7F8",
  "content_background": "#F4F8FA",
  "primary": "#3E748D",
  "secondary": "#7B9FAE",
  "text": "#24313A",
  "metal": "#AABBC4",
  "pattern": "ripples",
  "symbol": "water"
}
```

## Тема 3: Воздух · Дыхание — `air`

**Ощущение:** свобода, ясность, пространство, движение мысли.

**Мотивы:** тонкие траектории, воздушные потоки, пунктирные орбиты и большие свободные поля. Не использовать буквальные облака, перья и крылья как основной визуальный язык.

```json
{
  "cover_background": "#DDE8ED",
  "cover_text": "#263741",
  "content_background": "#FAFCFD",
  "primary": "#507A8B",
  "secondary": "#8AA7B3",
  "text": "#27343B",
  "metal": "#9EADB4",
  "pattern": "currents",
  "symbol": "air"
}
```

## Тема 4: Огонь · Искра — `fire`

**Ощущение:** воля, энергия, проявленность, внутренний импульс.

**Мотивы:** расходящиеся лучи, искровые точки, линии тепла и концентрические вспышки. Не использовать реалистичное пламя и агрессивную красно-чёрную эстетику.

```json
{
  "cover_background": "#3A201D",
  "cover_text": "#FFF4E8",
  "content_background": "#FFF7EF",
  "primary": "#A94E37",
  "secondary": "#C8873F",
  "text": "#342622",
  "metal": "#C19B61",
  "pattern": "rays",
  "symbol": "fire"
}
```

`secondary` и `metal` являются декоративными токенами. Они не используются для длинного текста без отдельной проверки контраста.

## Экран выбора темы

Показывать пять вариантов:

- **Подобрать по карте** — система выберет визуальную тему автоматически;
- **Земля · Корни** — спокойное и собранное оформление;
- **Вода · Глубина** — мягкое и глубокое оформление;
- **Воздух · Дыхание** — светлое и просторное оформление;
- **Огонь · Искра** — тёплое и выразительное оформление.

Под селектором показать пояснение:

> Стихия влияет только на оформление книги. Она не меняет расчёты и содержание.

Нельзя маркировать одну тему как более точную, премиальную, женскую, мужскую, сильную или духовную.

## Правила бренда

1. Все четыре темы должны с первого взгляда считываться как один продукт.
2. Ни одна тема не получает дополнительные функции или более богатый layout.
3. Внутренние страницы преимущественно светлые; тёмная заливка допустима на обложке и разделителях.
4. Декор не должен конкурировать с текстом.
5. Element symbol и pattern создаются кодом из фиксированной библиотеки.
6. LLM не генерирует визуальные инструкции.
7. Название стихии не должно автоматически появляться в тексте книги.

## Почему не делать уникальный дизайн

Уникальная генерация дизайна для каждой книги приводит к нестабильной вёрстке, проблемам с PDF и контрастом, сложной визуальной QA, непредсказуемому бренду и невозможности массово исправлять уже созданные книги.

Индивидуальность продукта должна жить прежде всего в содержании, а не в бесконечном количестве декоративных исключений.
